PAPOX - What is this project about?

PAPOX stands for Policy Administration Point for OASIS XACMLv3. XACML is an eXtensible Access Control Markup Language, an OASIS standard to access control that can be used in any context. This project focus on the third version of XACML which is still a draft version.

Which are the main goals for this project?

  • To be able to store and retrieve policies from any source (eg.: xml database, xml file)
  • To retrieve information about Policy Decision Point's supported data types, combining algorithms and functions, so that the options available to administrator are only the ones that he can choosen
  • To have an interface that is usuable and that abstracts the complexity of XACML
