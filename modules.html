Policy Retriever modules

BerkeleyDBXmlPolicyRetriever

A module that uses Oracle's Berkeley DB Xml to store and retrieve policies.

Requirements

  • Oracle Berkeley DB XML installed;
  • Java Virtual Machine should be running in 32bits mode.

Information Retriever modules

InfoRetriever

A very simple module that reads a static xml file with a list of combining-algorithms, functions and data types.
